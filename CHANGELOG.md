# Change Log

All notable changes to the "vscode-floating-terminal" extension will be documented in this file.

## [1.0.0] - 2024-12-19

### Added
- 🚀 Initial release of Floating Terminal
- 🎯 Beautiful centered floating terminal window
- ⚡ Comprehensive Neovim-inspired keymaps
- 🎨 Stunning visual effects with blur and animations
- 📱 Multiple terminal support with easy switching
- ⚙️ Extensive configuration options
- 🔄 Status bar integration
- 🎪 Smooth animations and transitions
- 🔧 Auto-focus functionality
- 📋 Built-in keymap reference

### Features
- **Main Controls**: Toggle, create new, close current, close all terminals
- **Navigation**: Switch between multiple terminals with keyboard shortcuts
- **Customization**: Width, height, opacity, blur effects, animations
- **Terminal Settings**: Custom shell, startup directory, font settings
- **Visual Polish**: Modern design with rounded corners and glass effects
- **Developer Experience**: Hot-reload configuration, comprehensive documentation

### Keymaps
- `Ctrl+`` - Toggle floating terminal
- `Ctrl+Shift+`` - Create new terminal
- `Ctrl+Shift+W` - Close current terminal
- `Ctrl+Shift+Alt+W` - Close all terminals
- `Ctrl+PageDown/PageUp` - Navigate between terminals
- `Ctrl+Shift+,` - Open settings

### Configuration Options
- Appearance: width, height, borderRadius, opacity, blur, animation
- Terminal: shell, startupDirectory, fontSize, fontFamily
